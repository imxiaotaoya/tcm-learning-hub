# 中医方证互动传习系统 v0.2.0 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 tcm-learning-hub 从 mock 空壳改造为基于 nihaisha 真实数据的中医学习前端

**Architecture:** 方案A单体仓库。构建脚本解析 nihaisha Markdown → TypeScript 数据文件 → React 组件直接 import。截图只存索引不打包原图。

**Tech Stack:** React 19 + Vite 6 + TypeScript 5.8 + Tailwind CSS 4 + motion/react

- [ ] **Step 2: 写 buildData.ts — 核心解析器**

```typescript
// scripts/buildData.ts
// 解析 nihaisha references/*.md → src/data/*.ts
// 运行方式: npx tsx scripts/buildData.ts
```

解析 6 种 Markdown 格式：
1. `formula-patterns.md` — 表格行 → Formula[]
2. `lesson-map.md` — 课次表 + 摘要 → Lesson[] + Chapter[]
3. `screenshot-evidence.md` 等 11 个文件 → Screenshot[]
4. `six-channel.md` — 六经表格 → SixMeridianStage[]
5. `symptom-index.md` — 症状映射 → SymptomRoute[]
6. `bencao.md` / `acupuncture.md` — 药性和穴位数据填充

```bash

wc -l src/data/formulas.ts src/data/lessons.ts src/data/screenshots.ts src/data/herbs.ts
```
